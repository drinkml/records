
# Contributing to DrinkML Records

## Adding a new record
1. Fork this repo
2. Create your JSON file in the correct category folder
3. Validate against the DrinkML v1.0 spec
4. Open a PR with a brief description
5. A maintainer will review and merge

## Record naming
Files are named: `{drink-slug}@{version}.json`
Example: `negroni@v1.json`, `perpetual-ipa@2024.json`

## Editing an existing record
Records are immutable once merged. 
To update, create a new version: `negroni@v2.json`
